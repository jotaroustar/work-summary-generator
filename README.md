## 功能介绍

### 免费参考区
- **场景骨架速查**：覆盖 8 种主流场景（周报 / 月报 / 季报 / 年终总结 / 转正总结 / 述职报告 / 项目复盘 / 离职交接），一键查看标准骨架结构
- **量化表达工具箱**：强动词库、量化维度参考、STAR 四步法、反面示例，帮你把流水账变成有说服力的成果陈述

### AI 生成区
- **碎碎念速记**：直接输入散乱的工作内容，AI 自动归类整理并填入表单
- **渐进式引导表单**：根据总结类型动态展示字段，按需填写，不强迫
- **多维度配置**：总结类型、给谁看、岗位角色、输出风格，四个维度精准控制输出
- **防幻觉占位符**：用户未提供具体数据时，AI 留下高亮占位符而非自行编造，点击即可填入真实数字

### 输出区
- **自由编辑模式**：一键切换为可编辑文本框，直接手动微调
- **篇幅调整**：更精简 / 更详细，一键重新生成
- **换风格**：朴实务实 / 突出亮点 / 体制内正式，同一素材多种表达
- **历史记录**：本地保存最近 10 条，随时调取
